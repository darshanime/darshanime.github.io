In this post, we'll write our first scrapy program. But just before that, it is good to have a basic understanding of what happens when you visit any web-page using your browser. 


The browser is called the **client**. You enter the URL of the web-page you wish to visit and press go. The client sends the **server** hooked to that URL a **request**, asking for the page you wished. In response, the **server** sends back a **response** with the required content. This is pretty much all one needs to know to use Scrapy. Note the words in **bold** are how things are referred to by geeks officially.


What does the **request** that the **client** sends to the **server** look like ? What does the **response** look like ?

_______

{% highlight python %}

#Requests is a python library that helps us interact with the Internet
>>> import requests
>>> r = requests.get('https://www.google.com') 

#What the client sent to the server
>>> print r.request.headers
{'Connection': 'keep-alive', 'Accept-Encoding': 'gzip, deflate', 'Accept': '*/*', 
'User-Agent': 'python-requests/2.7.0 CPython/2.7.10 Linux/3.13.0-62-generic'}

#What the server sent back 
>>> print r.headers
{'content-length': '7803', 'x-xss-protection': '1; mode=block',
 'content-encoding': 'gzip', 'server': 'gws', 
 'cache-control': 'private, max-age=0', 
 'date': 'Sat, 05 Sep 2015 09:29:54 GMT', 
 'content-type': 'text/html; charset=ISO-8859-1', 
 'x-frame-options': 'SAMEORIGIN'}

{% endhighlight %}

_______


These dictionaries of information exchanged between the client and server are called **headers**. 

They are pretty self explanatory, we identify ourself in the request headers, say what type of data we wish, in what format etc. The server replies by defining some cookies, mentioning the time, date of the response and sending the requested data. This data is then parsed by the browser and displayed. 


Now with the basics out of the way, let's dive right into **Scrapy**.

I am assuming you have Scrapy installed on your systems. If not yet, go thru the official installation instructions [here](http://doc.scrapy.org/en/latest/intro/install.html){:target="_blank"}. 

_______

##Scrapy basics. 
<br>
All Scrapy projects have a basic structure whose main components are :

1. **Items**

We define an **`Item`** object and alert it about all the information we wish to extract from our webpage.

2. **Spider**

Spiders define how the site will be scraped i.e. how to perform the crawl, how to extract structured data from the pages (aka scraping).

3. **Pipelines**

After getting the data, we define what happens to it next here. We can clean it, store it in a database, send it for further processing etc.

The **`Hello World`** of scraping is getting the listings on the homepage of [`dmoz`](http://www.dmoz.org/){:target="_blank"}. Open the page in your browser, it looks like this : 

![dmoz webpage](/img/one.png "dmoz webpage")

In this demo, will extract and list the headings on the page i.e. Arts, Games, Kids and Teens etc. 


_______
<br>
Dump this into **`items.py`**:



{% highlight python %}

from scrapy import Item, Field

class Scrapy101Item(Item):
  title = Field()

{% endhighlight %}


We are simply telling scrapy what we wish to extract. We are defining a new **`Item`** class, which isn't really very different from the vanilla Python **`dict`**.  

Now, before we write our Spider, let's look at the HTML of the page to locate where our data lies in it. Use [`Firebug`](https://addons.mozilla.org/en-US/firefox/addon/firebug/){:target="_blank"} if you are on Firefox or the Dev Tools on Chrome. We are using these tools and not viewing the HTML directly because the page has a complicated structure (even for this relatively simple page) and to pinpoint the exact location which harbors our content is tiresome. 

_______
<br>
Move the cursor over the various **`div`** elements and expand the ones that have the target. Repeat until you reach it. We see that our titles are divided into three `div` classes each called **`one-third`**. 

![dmoz nav](/img/two.png "dmoz nav")

Copy their **`XPath`** by right-clicking.  

It looks like this :

**`/html/body/div[3]/div[3]/div[1]`**. 

Note this **`div`** has all the three `one-third` divs. We can further access them by appending **`div[1]`**, **`div[2]`** or **`div[3]`** respectively.


_______
<br>
We can now write our Spider. Navigate to the **`Spider`** dir and create a new file, I will name mine **`dmoz.py`**
 
Dump : 


{% highlight python %}

from scrapy.spiders import BaseSpider
from scrapy101.items import Scrapy101Item

class Scrapy101Spider(BaseSpider):
    name = "dmoz"
    allowed_domains = ["dmoz.org/"]
    start_urls = ["http://www.dmoz.org/"]
    
    def parse(self, response):
        for div in response.xpath('/html/body/div[3]/div[3]/div[1]/div'):
            item = Scrapy101Item()
            item['title'] = div.xpath('span/a/text()').extract()
            print item['title']

{% endhighlight %}



We named our spider **`dmoz`**, gave it a URL to start with and asked it to not go outside the **`dmoz.org`** domain. 

_______
<br>
The spider starts by crawling the URL in the **`start_urls`** list, i.e. it sends a **request** asking for this page and the **response** is given to the **`parse`** function. The response object has all the information regarding the crawl (URL of the page, the HTML, HTTP status etc). 

In our **`parse`** function, we iterate over each child of our **`/html/body/div[3]/div[3]/div[1]/div`** element (the three **`one-third`** divs). For each **`div`**, we will store the the text present in **`span/a/text()`** in the **`Scrapy101Item`**'s **`title`** field.

_______
<br>
Note that if we crawl now, we will get the output as : 

{% highlight python %}

[u'Arts', u'Games', u' and Teens', u'Reference', u'Shopping']
[u'Business', u'Health', u'News', u'Regional', u'Society']
[u'Computers', u'Home', u'Recreation', u'Science', u'Sports']

{% endhighlight %}


We get three lists with the titles in the them. We can access them using indexes or add another loop to get them in separate **`Items`** :


{% highlight python %}

from scrapy.spiders import BaseSpider
from scrapy101.items import Scrapy101Item

class Scrapy101Spider(BaseSpider):
    name = "dmoz"
    allowed_domains = ["dmoz.org/"]
    start_urls = ["http://www.dmoz.org/"]
    
    def parse(self, response):
        for div in response.xpath('/html/body/div[3]/div[3]/div[1]/div'):
            for entry in div.xpath('span'):
                item = Scrapy101Item()
                item['title'] = entry.xpath('a/text()').extract()
                print item['title']

{% endhighlight %}
