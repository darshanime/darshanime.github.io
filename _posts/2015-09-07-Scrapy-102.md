Up till now, what we did was elementary. It was not really useful and definitely not great fun. This time, let's follow some links, scrape multiple pages and get some real data. 

Our target will be [CarDekho](http://www.cardekho.com){:target="_blank"}. Okay, target is a strong word, we aren't really doing them any harm. In fact, let's promote them !

**Cardekho is a major online marketplace in the Indian car-resale industry. You can sell/buy all kinds of cars to/from dealers, individuals. The cars are sorted according to the city, price, kilometers traveled etc. It is cool :)**


Here is how their page looks like : 

**It will have changed by the time you look at it, don't be surprised.**
![cardekho webpage](/img/three.png "cardekho webpage")

_______
<br>
So, this is a good thriving business site, a great example of a typical scraping assignment your boss will ask you to execute. What do we wish to juice out ? Let's take all the cars up for sale in Mumbai ? If you make the necessary navigation, you will reach the [results](http://www.cardekho.com/used-cars+in+mumbai-all){:target="_blank"} page. There are **6,231** listings at the time of writing this post. 

We have to decide what information we need to take from the results. Let's say we wish the name of the car, the price and the kilometers traveled. 

_______
<br>
The results are spread across multiple pages. How do we follow the links to scrape all the pages ? 

Look at the URL (it is generally a great idea to do this). It reads : **`http://www.cardekho.com/used-cars+in+mumbai-all`**. 

Scroll down and navigate to the next page, the URL changes to : **`http://www.cardekho.com/used-cars+in+mumbai-all/2`**. Note that clicking on the **`Last`** button and looking at the URL shows us that there are 156 pages in total, giving us about 40 entries on each page.

_______
<br>
All the pages will have similar structure and once we tell Scrapy where to pick the data from from page 1, we can repeat the procedure for all the pages. 

Use Firebug/Chrome DevTools to get the exact 'address' of the information to scrape. 

Like before, move the cursor on each element, expand the one which houses the desired part of the page and keep expanding until you reach the target. 

Copy the **`XPath`** of the element containing the listings as its children. Verify that in our case, it is :
**`/html/body/main/div/div[2]/div[2]/div[9]/form/ul/li`**. 

The parent **`li`** element has another 40 **`li`** elements as its children. They house all the information about the cars listed on that page. Navigate deeper to find the Title of the car. Verify that it is :

**`/html/body/main/div/div[2]/div[2]/div[9]/form/ul/li[1]/div[1]/div[2]/div[1]/a`**.

This is the absolute path. Relative to the parent **`li`** element above, it is :

**`div[1]/div[2]/div[1]/a`**

Similarly, the price : **`div[1]/div[3]/div[1]`** 
and the kilometers : **`div[1]/div[2]/div[3]/ul/li[1]/div[2]/span`**.

Before we code this up, let's test this in the scrapy shell which is very useful in making sure we have our **`xpath`**s right and that the output is as expected. 

Fire up the shell with our starting page :

**`$ scrapy shell "http://www.cardekho.com/used-cars+in+mumbai-all/" `**

This will open up the IPython shell with the **`response`** object ready for our queries. Let's see if we have the Titles of the cars right. 

**`response.xpath("/html/body/main/div/div[2]/div[2]/div[9]/form/ul/li/div[1]/div[2]/div[1]/a/text()")`**

This might be working for you. But guess what, I get nothing. This is because I have been redirected to a captcha page by Cardekho. You can view your response by 

**`In [1]: view(response)`**. 

This will open the received **`response`** in your default browser.

Open up **`items.py`** and dump this:

{% highlight python %}

from scrapy import Item, Field

class CardekhoItem(Item):
    title = Field()
    price = Field()
    distance = Field()

{% endhighlight %}


Next, write the **spider**. 

{% highlight python %}

from cardekho.items import CardekhoItem
from scrapy import Spider
from scrapy.http.request import Request

class CardekhoSpider(Spider):
    name = "cardekho"
    allowed_domains = ["http://www.cardekho.com"]
    start_urls = ["http://www.cardekho.com/used-cars+in+mumbai-all/"]
    
    #This is to not get redirected by CarDekho. We are identifying ourselves as a web-browser. 
    custom_settings = {'USER_AGENT' : 'Mozilla/5.0 (Macintosh; Intel Mac OS X 10_10_1) AppleWebKit/537.36 (KHTML, like Gecko) Chrome/40.0.2214.111 Safari/537.36'}
    
    def start_requests(self):
        #There are 162 pages, we are asking Scrapy to get us all of them.
        for i in range(162):
            yield Request("http://www.cardekho.com/used-cars+in+mumbai-all/" + str(i), self.parse)

    def parse(self, response):
        for sel in response.xpath('/html/body/main/div/div[2]/div[2]/div[9]/form/ul/li'):
            item = CardekhoItem()
            item ['title'] = sel.xpath('div[1]/div[2]/div[1]/a/text()').extract()
            item ['price'] = sel.xpath('div[1]/div[3]/div[1]/text()').extract()
            item ['distance'] = sel.xpath('div[1]/div[2]/div[3]/ul/li[1]/div[2]/span/text()').extract()                      
            yield item

{% endhighlight %}
